# 软件功能清单（internal-resource-system）

以下功能清单依据当前项目菜单、路由与后端 API 模块梳理。

## 1. 认证与权限

- **登录**：用户名/密码登录，获取 JWT
- **用户态**：
  - 退出登录
  - 前端自动携带 `Authorization: Bearer <token>`
- **权限控制**：
  - `ADMIN`：管理员权限（例如用户管理）
  - `EDITOR`：可编辑（新增/编辑/删除）业务数据

## 2. 首页

- **资源规模概览**：
  - 项目数量
  - 干系人数量
  - 计算资源数量

## 3. 项目资源管理

### 3.1 项目信息

- 项目列表
- 按条件查询（省份/地市/项目名称等，具体以页面为准）
- 新增/编辑/删除项目
- 查看项目详情

### 3.2 干系人

- 干系人列表（关联所属项目）
- 新增/编辑/删除干系人
- 查看干系人详情

## 4. 基础资源管理

### 4.1 远程管理

- 远程信息列表（关联所属项目）
- 新增/编辑/删除远程信息（Markdown/富文本内容）
- 查看远程信息详情（渲染展示）

### 4.2 计算资源（原“服务器”）

- 计算资源列表（按项目筛选、按 IP 筛选）
- 新增/编辑/删除计算资源
- 复制计算资源（以当前资源为模板，新建保存）
- 字段能力（以编辑页为准）：
  - IP / EIP / 操作系统类型/操作系统联动
  - CPU/内存/磁盘
  - 网络区域
  - 远程方式（关联远程管理信息）
  - 部署软件清单（关联软件清单）

## 5. 日常运维记录

### 5.1 服务器维护记录

- 列表（可按项目筛选）
- 新增/编辑/删除/查看
- 字段：
  - 项目（下拉关联项目）
  - IP（下拉关联计算资源，按项目联动）
  - 服务器维护类型（下拉，字典 `parentCode=000400`）
  - 操作内容、操作人、操作时间、备注
- 默认值：
  - 操作人：默认当前登录人
  - 操作时间：默认当前时间
- **说明**：该模块不写入“操作日志”

### 5.2 数据维护记录

- 列表（可按项目筛选）
- 新增/编辑/删除/查看
- 字段：
  - 项目（下拉）
  - 数据库（下拉关联数据库管理，按项目联动）
  - 需求来源（下拉，字典 `parentCode=000600`）
  - 是否合规（文本，提示：是否有盖章文件，没有则注明需求人姓名）
  - 处理措施（多行文本）
  - 处理数量（整数）
  - 处理时间（默认当前时间）
  - 处理人（默认当前登录人）
  - 备注
- **说明**：该模块不写入“操作日志”

## 6. 系统管理

### 6.1 软件清单

- 软件条目列表
- 新增/编辑/删除软件条目
- 查看软件条目

### 6.2 字典管理

- 字典树 + 字典列表
- 新增/编辑/删除字典项
- **说明**：字典表字段「字典类型」已从业务层面移除（数据库层需执行删除列 SQL）

### 6.3 用户管理（管理员）

- 用户列表
- 新增/编辑/删除用户
- 角色配置、启用/禁用等

### 6.4 操作日志

- 记录范围：项目/干系人/计算资源/远程信息/软件清单/字典/用户 的新增/变更/删除
- 可查询（起止时间/类型/对象/日志关键字）
- **说明**：日常运维记录模块不写操作日志
